jWhere

jWhere is a jQuery plugin useful to avoid the tedious task of using iterators to check conditions in a given set of elements.

How to use it

For example, imagine that you want to change the color of some <li> based on a regex matching its content. In traditional jQuery you should do something like:

var elements = [];
$("li").each(function (index, element) {
    if ($(this).html().match(new RegExp("Item \\d"))) {
        elements.push(this)
    }
});
$(elements).css("color", "red");

What you do with jWhere:

$("li").where({
    html: new RegExp("Item \\d")
}).css("color", "red");

What can jWhere do

With jWhere you can filter a set of DOM elements based on it's properties or what elements are near to it. Depending on what to check, you should use a pair of key/value or maybe evaluate arbitrary things using a anonymous function pattern. Check the following examples:

String

$("li").where({
    text: "Lorem ipsum"
});

Numbers

$("li").where({
    width: 200
});

Regexp

$("li").where({
    text: new RegExp(".*ipsum.*")
});

Function

$("li").where({
    text: function (value) {
        // non-true returns will be discarted in the filter
        return parseInt(value) > 10;
    }
});

Key/Something

$("li").where({
    css: {
        key: "display",
        value: "list-item"
    }
});
$("li").where({
    css: {
        key: "display",
        value: function (value) {
            return value !== "list-item";
        }
    }
});

Grouping

$("li").where([{
    text: function (value) {
        return parseInt(value) > 10;
    },
    css: {
        key: "display",
        value: function (value) {
            return value !== "none"
        }
    }
}]);
